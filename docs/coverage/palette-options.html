A · Strict

Maximum visual discipline. Reads as severe / austere / serious-intelligence. Cards are thin painted rectangles. Best for users who want zero ornament — closest in feel to the original Readout dossier.

B · Core + Depth + Texture

Same identity as A but cards feel material: brass-warm tint on highlighted surfaces, midnight-cool tint on quotes, film-grain texture, layered borders, soft inset highlight. Adds richness while keeping red as the only true accent.

C · Wider Editorial

Opens up the palette to ochre / blue / plum etc. Reads more like a print magazine layout. Risk: red stops being THE accent, and the page can start to look like a Bloomberg / FT layout.
